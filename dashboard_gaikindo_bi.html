Sumber Data
GAIKINDO (Gabungan Industri Kendaraan Bermotor Indonesia). Data historis wholesales, retail, produksi, ekspor, dan impor periode 2020–2025.
🔧
Teknologi
Power BI untuk visualisasi analitik. Website HTML/CSS/JS untuk integrasi dashboard. Data warehouse berbasis star schema (ETL pipeline).
